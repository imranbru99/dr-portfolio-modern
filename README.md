# 🩺 Dr. Portfolio Modern

A sleek, modern, and fully responsive portfolio template designed for physicians, medical researchers, and healthcare professionals. Built with a clean UI and smooth user experience to showcase expertise, publications, and achievements professionally.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/imranbru99/dr-portfolio-modern.git
```

### 2. Navigate to Project Folder

```bash
cd dr-portfolio-modern
```

### 3. Open in Browser

Simply open `index.html` in your browser.

## 🎨 Customization Guide

You can easily customize this template:

* ✏️ Edit **index.html** for content
* 🎨 Modify styles in **CSS files**
* ⚙️ Update interactions in **JavaScript files**
* 🖼️ Replace images inside `/assets/images`
* 🔗 Update social links and contact info
